## FAQ

**How long does each path take?**
Depends on your pace. AI Fluency can be completed in a few days. AI Product Thinking and Level Up AI are deeper — plan for a few weeks if working through all lessons.

**Do I get a certificate?**
No. These are open-source learning resources, not certification programs. The value is in the skills, not the paper.

**Can I skip to Path 2 or 3?**
Yes. Each path is self-contained. The journey (1 → 2 → 3) is recommended but not required.

**Why is this free?**
AI tools cost money. AI knowledge shouldn't. These paths are built by a practitioner who believes AI literacy is not a luxury good.
